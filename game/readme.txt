アクションゲームのシステムです。
コンパイルはDirectX8が入っているDirectX SDKが必要です。

左右の方向キーで移動します
zキーでリンゴを飛ばします。上キーを押しながらだと上方向にリンゴを飛ばせます。
xキーでジャンプ。敵をふんだ場合xキーを押した場合だと通常より高くジャンプできます。
敵はリンゴを当てるか、踏みつけることで倒せます。

マップ作成について

mapedit.exeでマップの作成および編集ができます。
マップエディタはconf.hのMAP_EDIT_MODEを1にしてコンパイルすることにより，マップエディタに
なります。

マップエディタは以下のように操作します。
読み込みのダイアログをキャンセル
マップは十字の赤がになってところが通行不可です。
十字の青は今選択している位置です。マップサイズは固定(100マスx15マスです)
で変更できません

キーボード
上下，左右キーで移動
q 保存
d マップを右に移動
s マップを左に移動
z 配置
x 種類選択
c 削除

a モード変更
*オブジェクト配置モード:
 プレイヤー，敵，アイテムの配置をできるモードです。
 プレイヤーはかならず1つしか置けませんかつ必ず配置
 しなければいけません。
 
*マップ配置モード:
f 壁のつうかできるようにする，もう一回押すと通行不可になる
t　実行/停止　編集したマップを実際にプレイする。
